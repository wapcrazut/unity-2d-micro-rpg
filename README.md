# Unity 2D Micro RPG

2D Micro RPG prototype made with Unity 2022 LTS.

![Preview](./preview.gif)

## Features

- 2D top-down movement and camera
- Player movement
- Enemy AI
- Interactable objects
    - Doors
    - Chests
    - Coins
- Combat
- Inventory
- UI and HUD
    - Health bar
    - Experience bar
    - Level
    - Inventory
    - Interactable object prompt

## Controls

- `WASD` or `Arrow keys` to move.
- `E` to interact with objects.
- `Left Control` to attack.
